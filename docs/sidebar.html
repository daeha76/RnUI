Sidebar

조합 가능한 사이드바 컴포넌트. Header, Content, Footer, Group, Menu 등 9개의 하위 컴포넌트로 구성됩니다.

shadcn/ui (React)와의 차이점

RnUI의 Sidebar는 React shadcn/ui와 API가 다릅니다. SidebarProvider, SidebarInset, SidebarTrigger 컴포넌트가 없으며, 상태 관리는 RnSidebar의 @bind-Open으로 직접 수행합니다. SidebarMenuSub, SidebarMenuSubButton, SidebarMenuSubItem 컴포넌트도 포함되지 않습니다.

Component Hierarchy

Sidebar를 구성하는 9개 컴포넌트의 계층 구조입니다.

RnSidebar — 최상위 사이드바 컨테이너 (<aside>) ├── RnSidebarHeader — 상단 헤더 영역 (<div>) ├── RnSidebarContent — 스크롤 가능한 콘텐츠 영역 (<div>) │ └── RnSidebarGroup — 메뉴 그룹 (<div>) │ ├── RnSidebarGroupLabel — 그룹 라벨/제목 (<div>) │ └── RnSidebarMenu — 메뉴 목록 (<ul>) │ └── RnSidebarMenuItem — 메뉴 항목 (<li>) │ └── RnSidebarMenuButton — 메뉴 버튼/링크 (<a>) └── RnSidebarFooter — 하단 푸터 영역 (<div>)

Components

RnSidebar

<aside>

최상위 사이드바 컨테이너. Open/Close 상태를 관리합니다.

RnSidebarHeader

<div>

사이드바 상단 헤더 영역. 로고, 앱 이름 등을 배치합니다.

RnSidebarContent

<div>

스크롤 가능한 주요 콘텐츠 영역.

RnSidebarFooter

<div>

사이드바 하단 푸터 영역.

RnSidebarGroup

<div>

메뉴 항목을 그룹화하는 컨테이너.

RnSidebarGroupLabel

<div>

그룹의 라벨/제목 텍스트.

RnSidebarMenu

<ul>

메뉴 항목의 목록 컨테이너.

RnSidebarMenuItem

<li>

개별 메뉴 항목 래퍼.

RnSidebarMenuButton

<a>

클릭 가능한 메뉴 버튼/링크. Href와 IsActive를 지원합니다.

API Reference

RnSidebar

사이드바의 최상위 컨테이너. <aside>로 렌더링됩니다.

Parameter Type Default Description
Open bool true 사이드바의 열림/닫힘 상태. @bind-Open으로 양방향 바인딩 가능.
OpenChanged EventCallback<bool> — Open 상태 변경 시 호출되는 콜백. @bind-Open 사용 시 자동 바인딩.
Width string "16rem" 사이드바 너비. CSS 단위를 포함한 문자열 (예: "16rem", "256px").
Class string? null 추가 CSS 클래스.
ChildContent RenderFragment? null 하위 컴포넌트 콘텐츠.

Method: Toggle() — Open 상태를 반전시킵니다. @ref로 참조하여 호출할 수 있습니다.

RnSidebarMenuButton

메뉴 버튼/링크. <a>로 렌더링됩니다.

shadcn/ui와 다른 점

React shadcn/ui의 SidebarMenuButton은 size, tooltip, asChild 등의 props를 제공하지만, RnUI에서는 Href와 IsActive만 지원합니다.

Parameter Type Default Description
Href string? null 링크 URL.
IsActive bool false 활성 상태 여부. true이면 bg-sidebar-accent 스타일이 적용됩니다.
Class string? null 추가 CSS 클래스.
ChildContent RenderFragment? null 버튼 콘텐츠 (아이콘, 텍스트 등).

RnSidebarHeader / Content / Footer / Group / GroupLabel / Menu / MenuItem

나머지 컴포넌트는 모두 동일한 기본 파라미터를 가집니다.

Parameter Type Default Description
Class string? null 추가 CSS 클래스.
ChildContent RenderFragment? null 하위 콘텐츠.

모든 컴포넌트는 [Parameter(CaptureUnmatchedValues = true)]를 통해 추가 HTML 속성을 지원합니다.

Usage Examples

Basic Sidebar

MySidebarLayout.razor
<div class="flex h-screen">
    <RnSidebar @bind-Open="_sidebarOpen">
        <RnSidebarHeader>
            <h2 class="text-lg font-semibold">My App</h2>
        </RnSidebarHeader>

        <RnSidebarContent>
            <RnSidebarGroup>
                <RnSidebarGroupLabel>Navigation</RnSidebarGroupLabel>
                <RnSidebarMenu>
                    <RnSidebarMenuItem>
                        <RnSidebarMenuButton Href="/" IsActive="true">
                            Home
                        </RnSidebarMenuButton>
                    </RnSidebarMenuItem>
                    <RnSidebarMenuItem>
                        <RnSidebarMenuButton Href="/settings">
                            Settings
                        </RnSidebarMenuButton>
                    </RnSidebarMenuItem>
                </RnSidebarMenu>
            </RnSidebarGroup>
        </RnSidebarContent>

        <RnSidebarFooter>
            <p class="text-xs text-muted-foreground">v1.0.0</p>
        </RnSidebarFooter>
    </RnSidebar>

    <main class="flex-1 p-6">
        @Body
    </main>
</div>

@code {
    private bool _sidebarOpen = true;
}

Toggle Sidebar with @ref

ToggleExample.razor
<RnButton OnClick="() => _sidebar.Toggle()">
    Toggle Sidebar
</RnButton>

<RnSidebar @ref="_sidebar" @bind-Open="_open">
    <RnSidebarContent>
        <RnSidebarGroup>
            <RnSidebarMenu>
                <RnSidebarMenuItem>
                    <RnSidebarMenuButton Href="/dashboard">
                        Dashboard
                    </RnSidebarMenuButton>
                </RnSidebarMenuItem>
            </RnSidebarMenu>
        </RnSidebarGroup>
    </RnSidebarContent>
</RnSidebar>

@code {
    private RnSidebar _sidebar = default!;
    private bool _open = true;
}

Setup

1. CSS Reference

App.razor
<link rel="stylesheet" href="_content/Daeha.RnUI/css/shadcn.css" />
CSS file name

CSS 파일명은 rnui.css가 아닌 shadcn.css입니다.

2. Import

_Imports.razor
@using Daeha.RnUI.Components.UI

3. Tailwind CSS Configuration

Tailwind CSS가 RnUI 패키지의 클래스명을 스캔하도록 구성합니다.

tailwind.config.js
module.exports = {
  content: [
    "./**/*.razor",
    "./node_modules/@anthropic-ai/**/*.js", // if applicable
    // RnUI package classes
    "./bin/**/Daeha.RnUI.dll"
  ]
}
