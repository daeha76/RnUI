Get Started

3단계로 RnUI를 프로젝트에 적용할 수 있습니다.

Step 1 — Install

NuGet 패키지를 설치합니다.

  • .NET 8.0 / 9.0 / 10.0 지원
  • Blazor Server & WebAssembly 호환
  • 외부 의존성 없음
Terminal 
dotnet add package Daeha.RnUI

Step 2 — Configure

Import와 스타일시트를 추가합니다.

  • _Imports.razor에 using 추가
  • App.razor에 CSS 링크 추가
  • Tailwind CSS 스캔 경로 설정
_Imports.razor 
@using Daeha.RnUI.Components.UI

Step 3 — Use

바로 컴포넌트를 사용할 수 있습니다.

  • 다양한 Variant와 Size 지원
  • @bind 양방향 바인딩
  • Class 파라미터로 스타일 확장
MyPage.razor 
<RnCard>
    <RnCardHeader>
        <RnCardTitle>Welcome</RnCardTitle>
        <RnCardDescription>Get started with RnUI</RnCardDescription>
    </RnCardHeader>
    <RnCardContent>
        <RnInput Placeholder="Enter your name" />
    </RnCardContent>
    <RnCardFooter>
        <RnButton>Submit</RnButton>
        <RnButton Variant="ButtonVariant.Outline">Cancel</RnButton>
    </RnCardFooter>
</RnCard>
