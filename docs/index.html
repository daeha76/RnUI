Get Started

3단계로 RnUI를 프로젝트에 적용할 수 있습니다.

Step 1 — Install

NuGet 패키지를 설치합니다.

  • .NET 8.0 / 9.0 / 10.0 지원
  • Blazor Server & WebAssembly 호환
  • 외부 의존성 없음
Terminal 
dotnet add package Daeha.RnUI

Step 2 — Configure

Import와 스타일시트를 추가합니다.

  • _Imports.razor에 using 추가
  • App.razor에 CSS 링크 추가
  • Tailwind CSS 스캔 경로 설정
_Imports.razor 
@using Daeha.RnUI.Components.UI

Step 3 — Use

바로 컴포넌트를 사용할 수 있습니다.

  • 다양한 Variant와 Size 지원
  • @bind 양방향 바인딩
  • Class 파라미터로 스타일 확장
MyPage.razor 
<RnCard>
    <RnCardHeader>
        <RnCardTitle>Welcome</RnCardTitle>
        <RnCardDescription>Get started with RnUI</RnCardDescription>
    </RnCardHeader>
    <RnCardContent>
        <RnInput Placeholder="Enter your name" />
    </RnCardContent>
    <RnCardFooter>
        <RnButton>Submit</RnButton>
        <RnButton Variant="ButtonVariant.Outline">Cancel</RnButton>
    </RnCardFooter>
</RnCard>

Component Examples

각 컴포넌트의 구체적인 사용법을 확인하세요.

Button

6가지 Variant와 8가지 Size를 지원하는 버튼 컴포넌트입니다.

  • Default, Secondary, Outline, Ghost, Destructive, Link
  • Xs, Sm, Default, Lg, Icon 사이즈
  • Disabled 상태 및 OnClick 이벤트
Button.razor 
<RnButton>Default</RnButton>
<RnButton Variant="ButtonVariant.Secondary">Secondary</RnButton>
<RnButton Variant="ButtonVariant.Outline">Outline</RnButton>
<RnButton Variant="ButtonVariant.Ghost">Ghost</RnButton>
<RnButton Variant="ButtonVariant.Destructive">Delete</RnButton>
<RnButton Variant="ButtonVariant.Link">Link</RnButton>

<RnButton Size="ButtonSize.Sm">Small</RnButton>
<RnButton Size="ButtonSize.Lg">Large</RnButton>
<RnButton Size="ButtonSize.Icon">🔔</RnButton>

<RnButton Disabled="true">Disabled</RnButton>
<RnButton OnClick="HandleClick">Click Me</RnButton>

Form Controls

Input, Textarea, Checkbox, Switch, Select, RadioGroup 등 폼 컨트롤을 제공합니다.

  • @bind 양방향 데이터 바인딩
  • 타입별 입력 (text, email, password, number)
  • Disabled 상태 지원
FormControls.razor 
<RnLabel For="email">Email</RnLabel>
<RnInput Type="email" Placeholder="you@example.com"
         @bind-Value="_email" />

<RnTextarea Placeholder="Write a message..."
            @bind-Value="_message" Rows="5" />

<RnCheckbox @bind-Checked="_accepted" />
<RnLabel>Accept terms</RnLabel>

<RnSwitch @bind-Checked="_darkMode" />
<RnLabel>Dark Mode</RnLabel>

<RnSelect @bind-Value="_fruit" Placeholder="Select...">
    <RnSelectItem Value="apple">Apple</RnSelectItem>
    <RnSelectItem Value="banana">Banana</RnSelectItem>
</RnSelect>

<RnRadioGroup @bind-Value="_plan">
    <RnRadioGroupItem Value="free" /> Free
    <RnRadioGroupItem Value="pro" /> Pro
</RnRadioGroup>

Card

Header, Title, Description, Content, Footer, Action으로 구성되는 카드 컴포넌트입니다.

  • Default / Sm 사이즈 지원
  • CardAction으로 헤더에 액션 추가
  • 유연한 컴포넌트 조합
Card.razor 
<RnCard>
    <RnCardHeader>
        <RnCardTitle>Notifications</RnCardTitle>
        <RnCardDescription>You have 3 unread messages.</RnCardDescription>
        <RnCardAction>
            <RnButton Size="ButtonSize.IconSm"
                      Variant="ButtonVariant.Ghost">⚙️</RnButton>
        </RnCardAction>
    </RnCardHeader>
    <RnCardContent>
        <p>Card content goes here.</p>
    </RnCardContent>
    <RnCardFooter>
        <RnButton>Save</RnButton>
        <RnButton Variant="ButtonVariant.Outline">Cancel</RnButton>
    </RnCardFooter>
</RnCard>

  • Table: Head, Body, Row, Header, Cell 구성
  • Avatar: 이미지 / Fallback 이니셜 / 그룹
  • Progress / Slider 바인딩

Tabs & Navigation

Tabs, Breadcrumb, Pagination으로 앱 내비게이션을 구성합니다.

  • Tabs: Default / Line 변형, 가로 / 세로 방향
  • Breadcrumb: Link, Separator, Page 조합
  • Pagination: @bind-CurrentPage 바인딩
Navigation.razor 
<RnTabs @bind-ActiveTab="_tab">
    <RnTabsList Variant="TabsListVariant.Line">
        <RnTabsTrigger Value="account">Account</RnTabsTrigger>
        <RnTabsTrigger Value="password">Password</RnTabsTrigger>
    </RnTabsList>
    <RnTabsContent Value="account">Account settings</RnTabsContent>
    <RnTabsContent Value="password">Password settings</RnTabsContent>
</RnTabs>

<RnBreadcrumb>
    <RnBreadcrumbItem>
        <RnBreadcrumbLink Href="/">Home</RnBreadcrumbLink>
    </RnBreadcrumbItem>
    <RnBreadcrumbSeparator />
    <RnBreadcrumbItem>
        <RnBreadcrumbPage>Current</RnBreadcrumbPage>
    </RnBreadcrumbItem>
</RnBreadcrumb>

<RnPagination @bind-CurrentPage="_page" TotalPages="10" />

Overlays

Dialog, AlertDialog, Sheet, Popover, Tooltip, DropdownMenu 등 오버레이 컴포넌트입니다.

  • Dialog / AlertDialog: @bind-Open 바인딩
  • Sheet: Top, Bottom, Left, Right 방향
  • Tooltip / HoverCard: Side 포지셔닝
Overlays.razor 
<RnDialog @bind-Open="_dialogOpen">
    <RnDialogHeader>
        <RnDialogTitle>Edit Profile</RnDialogTitle>
    </RnDialogHeader>
    <RnInput Placeholder="Name" @bind-Value="_name" />
    <RnDialogFooter>
        <RnButton OnClick="Save">Save</RnButton>
    </RnDialogFooter>
</RnDialog>

<RnSheet @bind-Open="_sheetOpen" Side="Side.Right">
    <RnSheetHeader>
        <RnSheetTitle>Settings</RnSheetTitle>
    </RnSheetHeader>
    <p>Sheet content here.</p>
</RnSheet>

<RnTooltip Side="Side.Top">
    <Trigger><RnButton>Hover me</RnButton></Trigger>
    <ChildContent><p>Tooltip text</p></ChildContent>
</RnTooltip>

<RnDropdownMenu Align="Alignment.End">
    <Trigger><RnButton>Menu</RnButton></Trigger>
    <ChildContent>
        <RnDropdownMenuLabel>Account</RnDropdownMenuLabel>
        <RnDropdownMenuSeparator />
        <RnDropdownMenuItem OnClick="Profile">Profile</RnDropdownMenuItem>
        <RnDropdownMenuItem OnClick="Logout">Log out</RnDropdownMenuItem>
    </ChildContent>
</RnDropdownMenu>

Accordion & Collapsible

접기/펼치기 컴포넌트입니다.

  • Accordion: 단일/다중 선택 모드 (Multiple)
  • Collapsible: 간단한 토글 패널
  • @bind 바인딩으로 상태 제어
Disclosure.razor 
<RnAccordion @bind-ExpandedItem="_expanded">
    <RnAccordionItem Value="item-1">
        <RnAccordionTrigger>Is it accessible?</RnAccordionTrigger>
        <RnAccordionContent>
            Yes. Follows WAI-ARIA design patterns.
        </RnAccordionContent>
    </RnAccordionItem>
    <RnAccordionItem Value="item-2">
        <RnAccordionTrigger>Is it styled?</RnAccordionTrigger>
        <RnAccordionContent>
            Yes. Uses Tailwind CSS for styling.
        </RnAccordionContent>
    </RnAccordionItem>
</RnAccordion>

@* 다중 선택 모드 *@
<RnAccordion Multiple="true">...</RnAccordion>

<RnCollapsible @bind-Open="_isOpen">
    <Trigger>
        <RnButton Variant="ButtonVariant.Ghost">Toggle</RnButton>
    </Trigger>
    <ChildContent>
        <p>Collapsible content here.</p>
    </ChildContent>
</RnCollapsible>

Utilities

Separator, AspectRatio, ScrollArea, Kbd, Empty 등 유틸리티 컴포넌트입니다.

  • Separator: Horizontal / Vertical 구분선
  • AspectRatio: 16:9, 4:3, 1:1 등 비율 제어
  • ScrollArea: 커스텀 스크롤 영역
Utilities.razor 
<RnSeparator />
<RnSeparator Orientation="Orientation.Vertical" />

<RnAspectRatio Ratio="16.0/9.0">
    <img src="photo.jpg" class="object-cover w-full h-full" />
</RnAspectRatio>

<RnScrollArea Class="h-[200px] rounded-md border">
    <p>Scrollable content...</p>
</RnScrollArea>

<p>Press <RnKbd>Ctrl</RnKbd> + <RnKbd>S</RnKbd> to save</p>

<RnEmpty>
